Los Selectores en CSS

¿Qué son los selectores?

Los selectores CSS son instrucciones que nos permiten identificar a qué elementos HTML queremos aplicar estilos. Con ellos, podemos cambiar colores, tamaños, márgenes y mucho más.

¿Cómo se usan en el código?

La forma básica de usar un selector consiste en escribir su nombre, abrir llaves {} y dentro colocar las propiedades de estilo. Ejemplo: 


h1 {
  color: navy;
  font-weight: bold;
}

Tipos más comunes de selectores (sin tabla)

Selector de Etiqueta

Selecciona todos los elementos de un tipo. Ejemplo: p afecta a todos los párrafos.

Selector de Clase

Aplica estilos a elementos que tengan una clase específica. Ejemplo: .importante.

Selector de ID

Se usa para un solo elemento con un ID único. Ejemplo: #menu.

Selector Universal

Aplica estilos a todos los elementos. Ejemplo: *.

Selector Descendiente

Aplica estilos a un elemento dentro de otro. Ejemplo: div p afecta a párrafos dentro de divs.

Selector de Hijo Directo

Afecta solo a hijos directos de un elemento. Ejemplo: ul > li.

Selector por Atributo

Selecciona elementos según atributos específicos. Ejemplo: input[type="text"].

Pseudoclases

Aplica estilos en ciertos estados. Ejemplo: a:hover cuando el mouse pasa sobre un enlace.

Pseudoelementos

Permiten modificar partes de un elemento. Ejemplo: p::first-letter estiliza la primera letra de un párrafo.

Ejemplos aplicados

  • h2: Cambiar el color de todos los subtítulos.
  • .destacado: Resaltar mensajes importantes.
  • #pie: Personalizar el footer de la página.

Dudas frecuentes

¿DATOS ADICIONALES?

Sí, puedes crear selectores combinados como div.alerta o #contenido p.

¿QUÉ SELECTOR TIENE MÁS PRIORIDAD?

En orden de mayor a menor prioridad: ID, Clase, Elemento, Universal.
